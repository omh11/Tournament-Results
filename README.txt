Project: Tournament Results  - Omar Hiari
================================

Required Libraries and Dependencies
-----------------------------------
Project requires Python v2.*, vargrant v1.7.4, and VirtualBox 5.0.10 to be installed


How to Run Project
------------------
From command line do the following:
- cd into /vagrant/tournament
- Use the command "vagrant up" to power on the virtual machine 
- Use the command "vagrant ssh" to log into the virtual machine
- Execute the command "psql" 
- Execute the command "\i tournament.sql"
- Execute the command "\q"
- Execute the command "python tournament_test.py"
